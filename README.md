# TeleportMod v1.4.0 (Minecraft 1.21.X)

Fabric-мод для быстрого управления командами телепортации и варпами, добавляющий `/tpa` и макросы.

⚠ **Требование:** **Fabric API**.

---

## 🕹 Управление (Версия 1.4.0)

* **Двойной клик:** Базовые команды (O, R, C) активируются двойным нажатием (400 мс).
* **Безопасность:** Автоматическая блокировка макросов при открытом чате/инвентаре.
* **Уведомления:** Статусы отображаются в экшн-баре.

### Горячие клавиши

| Клавиша | Команда | Функция |
| :--- | :--- | :--- |
| **O** / **Щ** | `/home o` | Быстрый телепорт (переименуйте дом в "o"). |
| **R** / **К** | `/rtp` | Рандомный ТП (кулдаун 20с). |
| **C** / **С** | `/spawn` | Мгновенный спавн. |
| **P** / **З** | `/warp` | Меню варпов (удержание) / Быстрый ТП (клик). |
| **B** / **И** | `/tpa` | **[NEW]** Интеллектуальный TPA-движок. |

---

## 🗺 Система Варпов (P)
* **Зажатие:** Выбор варпов стрелками.
* **Двойной клик:** ТП на первый варп.

---

## 👥 TPA-движок и трекер (B)
* **Клик:** Статус игрока (3 сек).
* **Зажатие (>300мс):** Список игроков (выбор стрелками).
* **Двойной клик:** Отправка `/tpa` / `/tp`.
* **Авто-сканер:** Очистка ников от префиксов (включая `[Админ]`).

---

## ⚙ Настройки (M)
* Зажмите **M** на 3 секунды для открытия меню.
