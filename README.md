# TeleportMod v1.3.6 (Minecraft 1.21.X)

Мод для Minecraft, который привязывает популярные команды телепортации к горячим клавишам.

> ⚠️ **Важное требование:** Для работы мода обязательно требуется [Fabric API](https://modrinth.com).

## 🕹️ Управление и особенности

* **Активация двойным нажатием:** Все указанные ниже команды срабатывают только при **двойном нажатии** на соответствующую клавишу.
* **Просмотр инструкции:** Вы всегда можете перечитать это руководство прямо внутри игры в меню настроек мода.

### Горячие клавиши и команды

| Команда | Клавиша (ENG / РУ) | Описание |
| :--- | :---: | :--- |
| `/home o` | **`O`** / **`Щ`** | Телепортирует на точку дома **o**. *(Примечание: для работы необходимо переименовать/создать точку дома с именем **O**).* |
| `/rtp` | **`R`** / **`К`** | Случайная телепортация. Перезарядка по умолчанию — 20 секунд *(можно изменить в меню `M`)*. |
| `/spawn` | **`C`** / **`С`** | Телепортирует вас на спавн мира. |
| `/warp` | **`P`** / **`З`** | Телепортирует на определенный варп *(по умолчанию настроена деревня)*. |

---

## 🗺️ Меню выбора варпов

* **Открыть меню:** Зажмите и удерживайте клавишу **`P`**, чтобы появилось меню выбора точек телепортации.
* **Телепорт:** Отпустите клавишу **`P`**, чтобы мгновенно переместиться на выбранный варп.
* **Управление:** Вы можете удалять и добавлять новые варпы прямо в настройках мода.
* **Навигация:** Используйте стрелки **Влево** и **Вправо** для точного удаления варпов.

---

## ⚙️ Настройки и конфигурация

* **Открыть настройки:** Зажмите и удерживайте клавишу **`M`** в течение **3 секунд**, чтобы открыть меню управления.
* **Навигация:** Используйте стрелки **Вверх** и **Вниз** для перемещения по пунктам меню.
* **Подтверждение:** Нажмите **`Enter`**, чтобы выбрать или подтвердить действие.
